# 🛒 E-Commerce Data Analysis using SQL

## 📌 Project Overview
This project analyzes an e-commerce dataset using SQL to extract insights on user behavior, product performance, sales trends, and overall business performance.

---

## 🗂️ Database Structure
The database consists of the following tables:

- **users** → Customer information  
- **orders** → Order details and transaction data  
- **product** → Product information and pricing  
- **order_items** → Items included in each order  
- **reviews** → Customer feedback and ratings  
- **events** → User activity tracking  

---

## 🛠️ Tools & Technologies
- SQL  
- MySQL  

---

## 📊 Key Analysis Performed

### 🔹 User Analysis
- Total users and gender distribution  
- User activity based on events  
- Users with high spending and engagement  

### 🔹 Sales & Order Analysis
- Total orders and revenue (completed orders)  
- Orders by status  
- Orders above average value  

### 🔹 Product Analysis
- Most purchased products  
- Revenue by category  
- Products never ordered or reviewed  
- Product ranking based on price  

### 🔹 Customer Behavior
- Top customers by spending  
- Repeat customers  
- Users who never ordered  

### 🔹 Advanced Analysis
- Running total of revenue  
- Ranking products and users  
- Previous and next order analysis (LAG & LEAD)  
- First order per user  

---

## 🔍 Key Insights
- Identified high-value customers contributing most to revenue  
- Found top-performing products and categories  
- Detected inactive users and products with no sales  
- Analyzed user behavior and engagement patterns  

---

## 🎯 Conclusion
This analysis helps understand customer behavior, optimize product strategy, and improve overall business performance using data-driven insights.
